## Get Started

required [task](https://taskfile.dev/installation/)

If not installed, see [Taskfile.yaml](./Taskfile.yaml) for completion

### 1. Container launch

```bash
task up
```

### 2. Enter the container

```bash
task exec -- app
```

### 3. Run the application

Server launches on http://localhost:8080

```bash
gradle bootRun
```
